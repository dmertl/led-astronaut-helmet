# Single Strip LED Astronaut Helmet

## Hardware

- Teensy 3.1
- Single Neopixel strip in helmet
- Button

### Configuration

- Button: pin 8
- LED strip pin: 10
- Helmet LEDs: 26

## Libraries

- Adafruit Circuit Playground
- Adafruit Neopixel
- Bounce2
- elapsedMillis
